    WSH Capitals Event Planner

    Using this Washington Capitals fan app, users can access information about player roster and a brief history of the team. This app accesses the ticketmaster.com API so users can purchase Capitals tickets through the site!



  • Repository: https://github.com/AKramer17/My-Movie-List

    My Movie List

    This app lets you create your personal movie list! Using the OMDB API, add movies to your list by using the search bar. Clicking on a movie will provide information such as the movie's rating, lead actors, Youtube trailer, and more.
